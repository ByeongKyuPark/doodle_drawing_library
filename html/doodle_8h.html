Objects revolve around the origin in a clockwise direction.Transformations apply to everything that happens after, and subsequent calls to the function accumulate the effect.Calling apply rotate(HALF PI) and then apply rotate(HALF PI) is equivalent to calling apply rotate(PI).The call to update window() resets all transformations. Brief desc.

Expandsand contracts vertices uniformly to increase or decrease a shape's size. Decimal percentages are used to specify the scale values.The function call apply scale(2.0), for example, increases the dimension of a shape by 200 % . Transformations affect everything that follows, and future calls to the function amplify the effect.For example, calling apply scale(2.0) and then apply scale(1.5) is equivalent to calling apply scale(3.0).Update window() calls reset†the transformation.

Brief desc. Sets the amount by which objects in the display window are displaced. The x parameter controls left / right translation, whereas the y parameter controls up / down translation. Transformations are cumulative and apply to everything that follows, and repeated calls to the function add up the effect.For example, using apply translate(50, 10) and then apply translate(20, 60) is equivalent to calling apply translate(70, 70).When the update window() function is run, all transformations are reset.The push settings() and pop settings() functions can be used to further customize this function.

Parameters
xleft/right translation
yup / down translation

noexcept

Brief desc.

The pop settings() function retrieves prior style settingsand transformations, whereas the push settings() function saves a new set of these settings. It should be noted that these functionalities are always utilized in tandem.They allow you to adjust the styleand transformation settingsand then revert to your previous settings.When you create a new state using push settings(), it builds on the current styleand transform information.To give more control, the push settings() and pop settings() functions can be incorporated. push settings() saves information about the current transformation's state and style settings (apply scale(), apply rotate(), apply translate(), apply matrix(),†set fill color()).
